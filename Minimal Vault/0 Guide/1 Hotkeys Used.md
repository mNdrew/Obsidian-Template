# ⌨ Hotkeys Used
---

| 👩‍💻 Command           | ⌨ Hotkey           |
|:----------------------- |:------------------ |
| 💠 Toggle Left Sidebar  | `ctrl + [`         |
| 💠 Toggle Right Sidebar | `ctrl + ]`         |
| 📏 Insert Template      | `ctrl + alt + W`   |
| 📌 Toggle Pin           | `ctrl + shift + P` |
| 📑 Cycle through panes  | `ctrl + tab`       |
| 🔴 Close Active File    | `ctrl + W`         |
|                         |                    |
